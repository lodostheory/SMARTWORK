사용 안내

사용 방법

  1. 소속 학교명을 입력합니다
  2. 출장신청서 엑셀 파일(.xlsx)을 업로드합니다
  3. 변환하기 버튼을 클릭합니다
  4. 생성된 여비정산서 ZIP 파일을 다운로드합니다

자동 처리 기능

  • 다수 출장자 일괄 처리 (최대 8건)
  • 직급별 운임 등급 자동 설정
    교장 → 1급 / 그 외 → 2급
  • 출장 일정 2행 자동 채움
  • 출장근거 / 출장목적 별도 행 분리
  • 13행 × 10열 새 서식 Word 생성
서버 실행 필요

이 기능을 사용하려면 먼저 터미널에서:

pip install flask openpyxl python-docx python app.py
